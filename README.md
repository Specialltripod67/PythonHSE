# Задание 1
## Задача 1
Узнаём длину слова

Выясняем чётное или нечётное 

Если слово чётное выводим 2 центральных символа, а если нечётное то 1 центральный символ
## Задача 2
Создаём функцию в которой: 

Проверяем равенство списков

Если не равны списки то выводится предупреждение об этом

Если списки равны то происходит сортировка списков по алфавиту и выводятся пары
